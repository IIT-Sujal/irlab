mukhyavArtAH

7 pradhAnamantrI narendramodI bhArata-AsiyAna shikharopaveshanAt‌ prAk‌ adya
navadillyAM AsiyAna-deshAnAM pramukhaiH sArdhaM dvipakSika-sambhASaNaM vidhAsyati|

 nirvAchnAyogena tripurA-vidhAnasabhA-nirvAchnArthaM adhisUchnA prasAritA|

pradhAnamantrI

bhArata-AsiyAna shikharopaveshane bhAgaM nirvoDhuM AsiyAna-deshAnAM pramukha-netAraH
rAjadhAnIM navadillI samprAptavantaH| ete sarve gaNatantradivasa-samArohe mukhya-
atithitayaH api santi| eteSu viyatanAma-kamboDiyA-philIpinsa-myAMmA-siMgApura-
thAIlaiNDa-maleshiyA-lAosa-brunaI dArUsalAma-inDoneshiyAyAH pramukhAH vartante|
pradhAnamantrI etaiH sAkaM dvipakSika-vArtAM sametya AtaMkavAda-viSaye5pi chrchA
vidhAsyati|

lAlU-sIbIAI

rAMchyAH sI. bI. AI. nyAyAlayena rASTriya-janatA-dala-pramamukhaM lAlU yAdavaM
sametya bihArasya pUrvamukhyamantrI jagannAtha-mishna-loka-lekhA-samityAH pUrva-adhyakSa-
jagadIsha-sharmA ch chADaibAsA gau-grAsa atha ch koSAgArAt‌ sapta-trishat‌-koTyadhika-
avaidha-dhana-rAsheH niSkAsana-prakaraNe paMch-paMch-varSAtmakaH kArAgAra-daNDaH atha ch paMch-paMch
lakSAtmaka-daNDa shulkaH shrAvitaH|

Apa-vidhAyakAH

Ama-AdamI-dalasya vishati-vidhAyakAnAm‌ AyogyatAM virUddhaya vAdashravaNe
dillI-uchch-nyayAlayaH kendrasya adhisUchnAM na sthagitavAn| paraM nyAyAlayena
nirvAchnAyogaH AdiSTaH asti yat‌ asau janavarI UnatriMsha-divasaM yAvat rikta-padeSu
nirvAchna-tithi na ghoSayet‌|

tripurA

tripurA vidhAnasabhA nirvAchnAnAM kRte adhisUchnA prasAraNAnantaraM nAmAMkana-
patrANAM prapUraNa-kArya prArabdham| etadarthaM pravartamAna mAsasya antimo divaso
vinirdhAritaH| tatra SaSTi-sadasyAtmikAyAH vidhAnasabhAyAH kRte pharavarI mAse aSTAshe
dinAMke matadAnaM bhavitA| matAnAM gaNanA ch mArch tRtIyAM bhaviSyati|

nirvAchnAyogaH

navachitena mukhya-nirvAchnAyuktena o. pI. rAvatena bhavitaM yat AyogaH
nirvAchna-prakriyAyAM divyAMgAnAM sakriya-bhAgaM sunishchetuM kArya kartavyam| nirvAchna-
prakiyAyAM divyAgAM-samAveshana-viSayamAlakSya navadillyAM samAyojite ekasmin
antArASTriya sammelane tenodIrita yat lokatAntrika-prayogAya divyAMgAnAM
sashaktikaraNaM-prAshAsanika-gatividhisu-ch teSAM samAveshana sahAyI siddho bhaviSyati|

bAlikA-divasaH

adya rASTriya-bAlikA-divaso'sti| divaso'yaM bAlikAnAM kRte sama-avasarAn
sunishchetuM dhyAvadhAnArthaM AmAnyate| avasare'smin pradhAnamantriNA narendra modinA
proktaM yat vibhinneSu kSetreSu bAlikAnAm upalabdhisu deshaH gauravam anubhavati| sUchnA
prasAraNa mantriNA smRtiIrAnI varyayA bhaNitaM yat bAlikAnAM sashaktikaraNe samAna-
avasarAH bhavitavyAH| rASTriya-stare eka putrI kRte indirA-gAdhI-chAtravRtti-yojanA api
prachAlayiSyati|

sImA-shulka

eSamaH sImA-shulka-kendrIyotpAda-kara-vibhAgasya chtushchtvArishad‌ karmakarAH
vishiSTa-sevArtha rASTrapateH pramANa-patreNa samAjayiSyanate| karmakarANAM vividha kSetreSu
kArya-niSpAdanAdhAreNa rASTrapateH pramANa-patraM dIyate, gaNatantra-divasAt pUrvamasya
udghASaNA ch kriyate|

uttama-jIvana-rakSA-padaka

gujarAtasya basachAlakaH shekha salima gaphuraH shauryArthaM rASTrasya dvitIya-sarvochch-
nAgarika-sammAnena sabhAjayiSyate| AtaMvAdInAM golikAH avigaNayya gaphureNa dvi-
paMchAshat-amaranAtha-yAtrINAM jIvanaM rakSitam AsIt| gaNatantra-divasAt pUrva gRha-
mantrAlayena tasmai uttama-jIvana-rakSA-padakam udghoSita masti|

pAka-mRtAH

pAkistAne aphagAnistAnasya sImni amerikA-dvArA kRte'bhiyAne hakkAnI-
gulmasya ekaH shIrSaH senApatiH sva-sahayogishSyAM saha mRtaH| mAdhyamaiH sUchyate yat
hakkAnI-senApatiH ehasAn utavA khavArI sva-sahayogibhyAM saha chAlaka-rahita-
vimAnasya AkramaNe paMchtatvaMgataH|
