pradhAnamantrI

pradhAnamantriNA narendramodinA adya rAjasthAne bAimerajanapadasya pachpadrAyAM
rAjyasya prathama-tailapariSkaraNa-pariyojanAyA shubhArambho vihitaH| trichtvArishat‌-sahasa-
koTi-rUpyakeSu eSA pariyojanAH hindustAna-peToliyam‌ iti kendrIyopakramasya rAjya-
sarvakArasya ch saMyuktodyamo vartate, yA hi varSa-chtuSTaye taila-pariSkaraNaM kariSyati|
anena rAjyasya chtustrishat‌-sahas-koTi- rUpyakANAm‌ atiriktAyo bhaviSyati| api
ch sahasrashaH janAnAM kRte pratyakSa-parokSa-niyukti-avasaro lapsyate| avasare'smin
pradhAnamantriNA ukta yat‌ pariyojanayA rAjyasya janAH sukha samRddhi ch prApsyanti|
atrAvasare peTroliyam mantrI dharmandra-pradhAnaH rAjyasya mukhyamaMtrI vasundharA rAje api
sambodhitavantau|

uchchtamanyAyAlayaH

uchchtama~nyAyAlayAnusAra yUnAm‌ antarjAtIya-vivAha-sambandho sarvaiH svIkAryaH|
khApa-panJchAyataiH athavA anyaiH kaishchidapi asya virodho naiva yuktaH| kendra-prashAsanaM
nyAyAlaya-bandhunA pradattaparAmarshAn‌ AdhRtya uttara pradAtumapi vinirdiSTam‌| sva-
parAmarshe nyAyAlaya-bandhunA Ihak‌-navavivAhitAnAM dampatInAM pIDanaM hatyAdi nirodadhum‌
upAyAn vidhAtuM sarvakAraH niveditaH AsIt|


nAgara-vikAsamantrI haradIpasiha-purI nigadati yat drutagatyA nagarIkaraNena deshaH
paryAvaraNapratikUlatAmanubhavati| prashAsanaM etAH samasyAH nirAkartu prayatnarataM vidyate|
asau adya navadillyAM smArTa siTIja iti sammelane sambhASate sma|

uchchtamanyAyAtrayaH

uchchtamanyAyAlayena mahArASTprashAsana samAdiSTa yat‌ sIbIAI-nyAyAdhIshasya
bI.ech.loyA ityasya mRtyu-sambaddha-patrANi yAchikAkartRbhyaH samupalabdhaM kArayet
iti| yAchikAkartAraH tasya mRtyoH svatantrAnvIkSaNaM yAchnte|

rASTrapatiH

rASTrapatinA rAmanAtha-kovindena deshe chikitsA-visheSajnJAnAM mahati AvashyakatA
pratipAditA navadillyAM bhAratIya-AyurvijnJAna-saMsthAnasya panJchchtvAriMshat-
dIkSAntasamArohe sambhASamANena rASTrapatinA uktaM yat chikitsakAH
atyadhikakAryabhAreNa nimajjitAH santi|

pravartananideshAlayaH

pravartananideshAlayena dhanashodhanaprakaraNe rASTriya-janatA-dalAdhyakSasya
lAlUprasAdasya aparo jAmAtA rAhulayAdavaH paripRchchArtham AhUto'sti| avadheyamasti
yat asminneva prakaraNe anyaiH saha mIsAbhAratI-dampatiH api pUrvameva paripRchchArthaM
samAhUtaH AsIt‌|

rohigyA-bAgladesha

bAgladeshena samAshvAsitamasti yat‌ rohigyAjanAnA myAmAdesha prati
punarpraSaNaprakriyA varSadvayAbhyantare samApayiSyate| myAmAdeshasya rAjadhAnyAM naipyATA-
ityatra deshadvayoH upaveshanAnantaraM adya bAMglAdeshasya videshamantrAlayena vaktavye
etat prakAshitamasti|

dhIvarAH

shrIlaMkA-nausenayA SoDasha-bhAratIyadhIvarAH chturnaukAbhiH saha nigaDitAH| sarve'pi
ime shrIlaMkA-jalakSetre matsyamAraNaM kurvanto Asan| anena sahaiva shrIlaMkAyAM bhAratIya-
dhIvarANAM saMkhyA saMvardhya trayodashottaraikashataM sanJjAtA asti|

AkAshavANI
AkAshavANyAH samAchAra-sevA-prabhAgenAd‌ya navadillyAM ilekTonika apashiSTa-
prabandhanamAdhRtyaikA kAryashAlA samAyojitA| kAryashAlA sambodhayan‌ visheSajnJH
rAmanAtha-khannA asUchyat yat vigata panJch-saptavarSeSu eteSAm apashiSTAnAm mAtrA
dviguNitA sanJjAtA| deshasya kRte chintanIyA eSA sthitiH vartate|
