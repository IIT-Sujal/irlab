sAyam‌ - 06H10
“gaNatantra divasopalakSya bhavadbhayo shubhakAmanAH”

7 rASTramadya navaSaSTitamaM gaNatantradivasam sammAnayati| pramukhasamArohaH rASTriyarAjadhAnyAH
rAjapathe samAyojitaH|

7 desha-videsheSu api gaNatantradivasa-samArohAyojanasya sUchnA|

7 pradhAnamantriNaH narendramodinaH adya maleshiyA-lAosa iti deshadvayorpradhAnamantribhyAM sArdhaM
dvaipAkSikamupaveshanam|

7 dakSiNakoriyAyAH chikitsAlaye sahasA pradIpta-agnikANDe eka chtvAriMshajjanAnAM mRtyuH|

gaNatantradivasaH

rASTramadya navaSaSTitamaM gaNatantradivasaM AmAnayati| samArohasya pramukha-kAryakramaH rASTriya-
rAjadhAnyAM rAjapathe samanuSThitaH| yatra rASTrapatinA rAmanAtha-kovindena dhvajottolanaM vidhAya
sainyapraNAmAH svIkRtAH| aiSamo varSe AsiyAnadeshAnAM dasharASTrAdhyakSAH mukhyAtithitvena
virAjamAnAH Asan| kAryakramAt prAk pradhAnamantriNA senApramukhatrayaM sametya amara-javAna-
jyoti-smArake shraddhAnJjaliH samarpitaH| samArohAvasare rASTrapatinA rAmanAthakovindena
shAntikalAyAH sarvochch-vIratA-puraskAraH ashoka-chkraH vAyusenAyAH garUDa-kamANDo-kArporala
jyoti-nirAlAyai maraNoparAntaM pradattaH| sammAnamidaM tasya patrI-mAtrAbhyAM chAdhigatam‌| asau
jammUkashmIre AtaGkavAdidvayaM vyApAdya vIratAM gatamAsIt|

pradhAnamantrI

pradhAnamantriNA deshavAsibhyaH gaNatantradivasasyopalakSye shubhakAmanAH vyAhRtAH| atrAvasare
samAgatebhyaH AsiyAna-rASTrAdhyakSebhyo'pi amunA dhanyavAdo vijnJApitaH| shrImodinA bhaNitaM yat
avasaro'yam asAdhAraNaM, AsiyAna-pramukhANAM yAtreyaM aitihAsikI vidyate| sUchnAprasAraNa-
mantriNA smRti irAnI-varyeNApi shubhakAmanAH vyAhRtA| anayA proktaM yat janAH rASTrasya
upalabdhiSu garvamanubhUya nUtana-bhArata-nirmANe yogadAnArthaM saMkalpaM kartavyam|
gaNatantradivasopalakSyo kAMgresAdhyakSeNa rAhulagAndhinA api shubhakAmanAH vyAhRtAH, saMvidhAna-
saMrakSaNa-saMkalpo karaNIyaH iti janAH prodIritAH| gaNatantradivaso'yaM videsheSu api sollAsena
AmAnyate|

pradhAnamaMtrI upaveshanam‌
pradhAnamantrI narendramodI lAosadeshasya pradhAnamantriNA sArdhaM adya daipAkSikamupaveshanaM
karoti| atrAvasare ubhayordashayoH mithaH hita-saMrakSaNAdi-viSayaM sametya vibhinna-viSayeSu vichAra-
vimarsho abhUt| etadanantaraM pradhAnamantrI shrImodI maleshiyAyAH pradhAnamantriNA najIba-
rajjnJAkavaryeNa sAkamapi dvipakSIyamupaveshanaM vidhAsyati|

jammUkashmIraH

jammUkashmIraprashAsanena chtustriMshat -janebhyaH vividhakSetreSu pradatta-abhUtapUrva-yogadAnArtha
puraskArasyoddhoSaNA vihitA| eteSu shikSAkSetraM prAshAsanikakaSatraM, kalAkSetraM prasArajagachr‌
sammilitAni santi| shrInagarasya samAchArapramukhAya mushtAkra-ahamadAya api tatkaSetre pradatta-
vishiSTa-yogadAnArtha puraskAro'yaM pradIyate|

dakSiNakoriyAyAH jiyoMgasAMga prAntasya miriyAMganagare adya prAtaH kasmishchit
chikitsAlaye sahasA pradIpte agnikANDe ekachtvAriMshat-janAH mRtyupagatAH, aSTAshItyadhika-
janAshch vraNitAH jAtAH| vraNiteSu kechna gambhIrAH sUchyante| agnikANDa-kAraNAni anvIkSyante|

padmapuskAra-ghoSaNA

kendraprashAsanena gaNatantradivasa-samArohAt ekadivasapUrvaM padmapuraskArANAM ghoSaNA kRtA|
asmin avasare AsiyAna-saMghasya pratirASTram ekaiko janaH puraskAra-bhAjanI-bhUtaH| panJchAshItiH
janebhyaH padmapuraskArAH pradIyante| eteSu janatrayaM padmavibhUSaNena, navanAgarikAH padma-bhUSaNena,
trisaptati-janAshch padmashriyA sabhAjitAH santi| padmapuraskAra-prAptakartRSu SoDashajanAH vaideshikAH
vartante| gRhamantrAlayena proktaM yat Asmin varSe padmapuraskArANAM kRte sArdha-panJchdasha-sahasra-
Avedana-patrANi prAptAni Asan| saMskRtakSetre visheSa-ullekhanIya-yogadAnArthaM vAgyogatantrasya
pravartakaH naikeSAM durlabhagranthAnAM ch praNetA, mahAmanISI pro. bhAgIrathaprasAdatripAThI “vAgIsha
shAstrI” padmashrIH sammAnena sabhAjitaH asti|
