pradhAnamantrI

pradhAnamantriNA narendramodinA nepAlasya pradhAnamantriNA ke.pI.sharmA olinA ch saMbhUya nepAladeshe janakapurataH
bhAratadeshe ayodhyA-paryantaM basayAna-sevAyAH shubhArambhaH kRtavantau| avasare'smin shrI modI kathitavAn yat ubhau deshau
rAmAyaNa-paryaTana-sarkiTa’ ityasya vikAsAya kAryamAchriSyataH| tena janakapura vikAsAya shatakoTi rupyakANAM rAshi-
puTakaH udghoSitaH| bihAra-kAThamAMDU pradeshayoH janAn pArasparika bhadra-sauvidhyAya relayAna samparka sevA api
udghoSitA| etena deshayoH sambandhAH sadraDhayiSyante| pradhAnamantriNA tatra jAnakI-mandire pUjArchnApi sampAditA| tena proktaM
yat‌ nepAladeshaH asmAkaM 'prativeshi-prathama-yojanAyAM' sarvaprathamamAyati| tena pratipAditaM yat yadA “sabakA-sAtha-sabakA-
vikAsasya vArtA bhavitA tadA prativeshi deshAH api samAviSTAH bhavanti| nepAla deshasya-dvi-divasIya-yAtrAvadhau tatra
pradhAnamaMtrI tatratya shIrSanetRbhiH sambhASiSyate| shrImodinaH eSA tRtIyA nepAlayAtrA vartate|

pravidhi-divasaH

asheSa-desha adya rASTriya-pravidhi-divasaH AmAnyate| aSTanavattayuttara-navadasha-shata-tame varSa rAjasthAne pokharaNa-kSetre
prathama-saphala-paramANu-parikSaNo vihitaH| svIya mana-kI-bAta- kAryakrame sandarbhe'smin pradhAnamantriNA proktaM yat mahAtmA-
buddha-nAmnA vihitena parikSaNena dRSTaM yat shAntaye Abhyantarika- balamAvashyakaM bhavati| modinA punaH adhyarthitAH yat te
bhArataM AdhunikaM-shaktiyutaM-AtmanirbharAya pUrva pradhAnamantriNaH aTala- bihArI-vAjapeyI varyasya “jaya-vijnJAna’ mantraM
anupAlayeyuH| tena svIya-TvITa sandesheSu proktaM yat‌ prashAsanaM pravidhi-kSatre utkRSTa-anusandhAna-navAchAra-pragataye ch bahavaH
upAyAH vidadhAti| TvITa sandeshe rakSA mantriNA nirmalA sItAramaNenoktaM yat‌ rASTriya-pravidhi divasaH deshasya vaijnJAnika-
pravidhijnJAnAM kRte samarpitaH| atha ch rASTrapatinA rAmanAthakovindena navadillyAM samAyojite ekasmin kAryakrame bhaNitaM yat
pradUSaNa-samasyAH apAkaraNa-prayAsAntargataM pravidhi sudraDhIkaraNa-nUtana-sImAH anveSTumAvashyakatA vidyate| tena avasare'smin
pUrva rASTrapatiM abdula-kalAmapi smRtam|
