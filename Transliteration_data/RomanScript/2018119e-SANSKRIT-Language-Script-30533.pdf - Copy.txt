jammUkashmIraH

jammUkashmIre adya sagharSa-virAmollaghanaM kRtvA pAkistAna-senayA jammUkSetrasya triSu
sImAvarti-janapadeSu sImA-sthAtreSu ch golikA-prahArAH vihitAH| etaiH sImA-surakSA-balasya
arakSI nAgarikau ch mRtAH, unaviMshatishch AhatAH saMjAtAH| AhateSu balasya dau ArakSiNa
staH| araniyA-Ara. esa. purA, sAMbAyAH rAmagaDha-kaThuA-kSetreSu ch ete janAH hatAhatAH|
rajaurI-janapadasya nausherA-kSetre api saMgharSa-virAmamullaMghya pAkistAnena vidhvaMsaka-
golikAvarSaNaM kRtamiti sUchyate|

arUNa jeTalI

vittamantriNA arUNa-jeTalinA proktaM yat samavAyANAM lekhAsu pAradarshitayA chadma
samavAyaiH dhana-preSaNam avarodhuM shakyate| navadillyAM samavAya-saMvaddha-lekhA-sUchnAnAM veba-
phalake udghATayan tenodIritaM yat anayoH asmin kSetre samadhikA pAradarshitA bhaviSyati,
yA hi vyavasthAyAH kRte api samyak syAt|

sAmAjika-nyAyaH

sAmAjika-nyAya-adhikAritA mantrI thAvara-chndra-gahalotaH adya divyAgAnAM kRte shataM
veba-phalakAni prachAlitAni| saMprati divyAga-janaiH sukareNa prAshAsanika-vebaphalakAni
upayokSyante| “divyAMga-hitaiSI” nAmadheya-vebaphalakaM visheSatayA iSTi bAdhitAnAM kRte vidyate|
shrIgahalotena bhaNitaM yat nava-shatAni IdRshAni vebaphalakAni kramashaH prachAlayiSyante|

pradhAnamantrI

pradhAnamantrI-surakSita-mAtRtva-abhiyAnAntargataM koTyadhika-garbhavatI-mahilAnAM prasava-
pUrva-svAsthya-parIkSaNaM vihitam‌| mAtR-shishu-svAsthya-saMvardhanArthaM kAryakramo'yaMSoDashottara-dvi-
sahasratame varSa samArabdhaH| svAsthya-mantriNA je.pI.naDagA varyeNa sUchitaM yat‌ surakSita-mAtRtva॑
deshe sAmAjikAndolanatvam‌ abhajat| kAryakramAya aSTashatottra-chtussahasrAdhika-niji-
chikitsakaiH svaichchika-sevApradAtuM saMkalpaH kRto'sti|

mahilA-janayAtaH

bhAratIya nausenAyAH prathamA mahilA chAlikA dalIyaM sAmudrika-yAnaM AI. ena. esa.
vI tAriNI phAkalaiMisa-dvIpaM gachchan adya “kepa hArna' sthalaM pAritA| etadartha
pradhAnamantriNA narendramodinA-pratirakSAmantriNA nirmalA-sItAramaNanaH ch mahilA-chAlikAH
vardhApitAH|

saMyukta-rASTram‌

amerikA-deshena hAphija-saIdaM virUddhya vidherantargataM pUrNAbhiyojanArthaM samAhUtam‌|
amerikAyAH pratikriyA pAkistAnIya pradhAnamantriNaH sAhida-khAkka-abbAsinaH TippaNyAH
anantaram AgatA| tena pratipAditam AsIt yat saMyukta-rASTreNa ghoSitAtaMkIM saIdaM
virUddhya kimapi kartu na shakyate, yato hi tasya virUddhe ko'pi AropaH nAsti|
