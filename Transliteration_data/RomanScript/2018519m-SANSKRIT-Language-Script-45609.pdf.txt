pradhAnamantrI-jojilA

pradhAnamantrI shrInarendra-modI adya leha-kSetre shrInagara-leha-rAjamArge jojilA-suraGga-nirmANArthaM shilAnyAsaM vidhatte|
chturdasha-kIlomITara-dIrgho'yaM suraMgaH na kevalaM bhAratasya, api tu eshiyAyAH dIrghatamaH dishAdvaya-gAmI suraMgo bhavitA| anena
sarveSu RtuSu jammU-kashmIrasya shrInagara-kargila-leha-kSetrANi parasparaM sadA sampuktAni sthAsyanti, yAtrA-kAle ch mahallAghavaM
sampAdayiSyate|

karNATakam-adhiveshanam

karNATaka-vidhAnasabhAyAm adya aparAhNe chturvAdane yediyurappA-prashAsanasya shakti-parIkSaNaM bhaviSyati| etachch
gatadivase shIrSa-nyAyAlayena samAdiSTam| atrAntare, karNATaka-rAjyapAlaH shrIvajUbhAI-vAlA adya pUrvAhne ekAdasha-vAdanAt
rAjya-vidhAnasabhAyAH Ahnikam adhiveshanam AkAritavAn| atrAdhiveshane sadyo gRhIta-shapathaM yediyurappA-prashAsanam adya
sadana-paTale vishvAsa-matAni avAptuM prayatiSyate| rAjyapAlena bhAjapA-netA ke.jI.bopaiyA etadadhiveshane tAtkAlikIm
adhyakSatAM nirvoduM niyojito'sti| tataH purva, gatadivase shaurSa-nyAyAlayena yediyurappA-prashAsanaM nirdiSTaM yattat‌ adyaivAparAhle
vidhAnasabhAyAH paTale svIyaM mata-bAhulyaM saMsAdhayet| atredaM dhyAtavyamasti yad ataH pUrvaM rAjyapAlena yediyurappA-sannItaM
bhAjapA-vRndaM nUtana-prashAsana-saMrachnArtham AmantrayatA panJch-divasAbhyantare svIyaM mata-bAhulyaM sAdhayituM nirdiSTamAsIt|

shIrSanyAyAlayaH-karNATakam-kAryavAhyadhyakSaH

shIrSa-nyAyAlayaH karNATaka-vidhAnasabhAyAM bhAjapA-vidhAyakasya ke.jI.bopaiyA-varyasya kAryavAhi-adhyakSatvena
sadyo vihitAM niyuktiM virudhya kAMgresa-dharmanirapekSajanatAdalAbhyAm upa-sthApitAm Apatti-yAchikAm adya shroSyati| mukhya-
nyAyAdhIshena nyAyamUrttinA dIpaka-mishreNa prakaraNamidam nyAyamUrtteH e.ke.sikarI-varyasya adhyakSatA-yutaM tri-sadasyAtmakaM
nyAya-khaNDapIThaM prati vAda-shravaNArthamagresAritam‌| sandarbha'smin‌ vAda-shravaNam‌ adyAparAjnJe chturvAdane karNATaka-
vidhAnasabhAyAH paTale nirdhAritAt‌ bhAjapA-shakte-parIkSaNAt‌ pUrvam‌, pUrvAhne sArdha-dasha-vAdane vidhAsyate|

amerikA-TeksAs‌-golikAvarSaNam‌
amerikAyAH TeksAs‌-nagaryAM gatadine saiNTA-phe-vidyAlaye durApAdite golikA-varSaNe nyUnAnyUnAH dasha janAH mRtyaM
gatAH| mRtakeSu eko'dhyApakaH, nava chAtrAshch samAvishanti| etAn vyatirichya dashAnye chAtrAH kSatAH api saMvRttAH sUchyante|

vidyate| amerikIya-rASTrapatinA DAnalDa-TrampeNa hyastanIM ghaTanAmabhilakSya mahatI chintA vyaktIkRtA'sti|
