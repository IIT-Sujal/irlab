prAtaH- 06.55

mukhya-vArtAH

pradhAnamantrI narendramodI nigadati yat varSe'smin apraila-jUna-mAsayormardhya
sakalagRhayotpAda-vRddhimitau ApannAm‌ apachitim‌ apAkartuM prashAsanaM
pratibaddhamasti|

bhArata-jibUtI-deshAshyAm‌ AtaGkavAdaM virUdhya kaThora-padakSepArthaM sahamatiH

prakaTitA|

rASTranirmANAya yogadAnArtham atha ch ArthikAvasarAn‌ janayituM bhArata-ithiyopiyA-
deshAbhyAM sammAnArjanArthaM rASTrapatiH rAmanAtha-kovindaH bhAratIya-samudAyaM prAshaMsat|
kendrIya-gRhamaMtrI rAjanAtha-siMha aNDamAna-nikobAra-dvIpasamUhasya dvidivasIya-
yAtrArtham adya prasthAsyati|

pradhAnamantrI-arthavyavasthA

sakalagRhyotpAdasya vRhimitivaSaye prashAsanasya AlochnArthaM pradhAnamantriNA vipakSaH
dRDhatayA pratyutaritaH| tenodIritaM yat prAktan-prashAsanasya kAryakAle SaDvarSAvadhau aSTavAraM
vRddhimitiH pratishataM panJch-dashamalava-saptamitam ut vA ito'pi nyUnam abhilikhitam|
shrImodinA proktaM yat deshasya samakSam etAdRshyaH api paristhitayaH upasthitAH yadA Arthika-
vRddhimitiH apachIya pratishataM sArdhaika-mitA sanJjAtA|

nava dillyAM bhAratIya-kampanI sachiva-saMsthAnasya savarNajayantI samArohasya
udghATanAvasare shrImodI pratyapAdayat yat apachitiriyaM bhAratIyArthavyavasthAyai tarjanAkarI
arvatata yatohi asmin avadhau bhArataM uchch mudrAsphItimiteH atha ch vittIyahAneH sthitiM
sammukhIkaroti sma| pradhAnamantriNA proktaM yat avaidhana-samasyAM samAdhAtuM prashAsanena prathame eva
divase vishiSTAnvIkSaNadalaM saMghaTitam AsIt|

pradhAnamantriNA nigaditaM yat prashAsanena ArthikapariSkAra-dishi katipaya - mahatvapUrNa-
nirNayAH vihitAH santi atha ch prakriyeyam anuvartiSyate| shrImodinA vyApArisamudAyaH

samAshvAsitaH yat vastusevAkaravyavasthAyAH pravartana-kAraNAt samutpannAM bAdhAm apAkartuM
parivartanArthaM prashAsanaM sajjitamasti|

bhArata-jibUtI

bhArata-jibUtI-deshAbhyAM dIrghakAlika-dvipakSika-sambandhArthaM sahamatiH prakaTitA atha ch
deshadvayamadhye Arthika-vyAvasAyika-prAvidhika-shaikSika-vaijnJAnika-sAMskRtikakSetreSu sambandha-
sambardhanArthaM samIhA punarudIritA| rASTrapati-rAmanAthakovindasya dvidivasIya-jibUtI yAtrAyAH
dvitIye divase gatadine vyAhRte saMyuktavaktavye etat‌ pratipAditam‌|

rASTrapatinA rAmanAtha-kovindena jibUtI-rASTrapatinA ismAila-omara-gyulehena ch

jibUtI-deshasya sAmarika-mahatvam atha ch aphrIkA-kSetre shAnti-sandhAraNArtha bhAratasya bhUmikA
abhisvIkRtA| AtaGkavAdaH vaishvika-shAntaye sthiratAyai ch mahat-tarjanArUpA vidyate iti
viSaye netRdvayena vichAra-vinimayo vihitaH|

rASTrapati-ithiyopiyA

rASTranirmANAya yogadAnArtham atha ch ArthikAvasarAn janayituM bhArata-ithiyopiyA-
deshAbhyAM sammAnArjanArthaM rASTrapatiH rAmanAtha-kovindaH bhAratIya-samudAyaM prAshaMsat| gatadine
ithiyopiyA deshasya rAjadhAnyAm adisha-abAvA-nagaryAM bhAratIya samudAyasya svAgata-
kAryakramaM sambodhayatA| shrIkovindena Ayurveda-yogayoH prachArArthaM bhAratIya-samudAyaH kSAghitaH|

rAjanAtha-aNDamAnayAtrA

kendrIya-gRhamaMtrI rAjanAtha-siMha aNDamAna-nikobAra-dvIpasamUhasya dvidivasIya-
yAtrArtham‌ adya prasthAsyati| svIya-yAtrAvadhau tena vividha-vikAsa-pariyojanAnAM shubhArambho
vidhAsyate| AdhikArika-srotobhiH sUchitaM yat‌ gRhamantrI porTableyara-nagare pota-
pariyojanAyAH shilAnyAsaM vidhAsyati| tena digalIpure vividha-rASTiya-rAjapatha-
pariyojanAnAm‌ api shilAnyAso vidhAsyate|

smRtiH-aGkIyavyavasthA

sUchnA prasAraNamantrI smRti-IrAnI abravIt‌ yat‌ aGkIyavyavasthAyAM
rachnAtmakasampraSaNaM protsAhayituM prashAsanasya bhUmikA vartate| gatadine navadillyAm‌ avajarvara-

risarch - phAraNDeshana dvArA samAyojite kAryakrame asau abhibhASate sma|

                                                  iti vArtAH
