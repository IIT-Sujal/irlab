relamatrI

relamaMtrI pIyUSagoyalaH avAdIt‌ yat‌ ena.DI.e. prashAsanasya chturvarSe
relayAna-durghaTanAsu dviSaSTiH pratishata-paryantaM nyUnatA jAtA| tena proktam
yat relayAnAnAM surakSA prashAsanasya sarvochch prAthamikatA vartate| nija-
maMtralAyasya chturvarSasya upalabdhinAm viSaye adya navadillyAM vArtAharaiH saha
sambhASamANaH shrIgoyalaH asUchyat yat‌ vikAsArthaM AdhunikIkaraNArthaM ch
dIrgha-stare niveshaM abhavat‌| asmin‌ avasare relamaMtrI avAdIt‌ yat‌ rela-
madada-iti api ch menyU-oona-rela-iti chladUravANyAH navIna praNAlyAH api
nirmANaM abhavat| relamaMtrI yaH kRSNAGgAra-maMtrAlayasya kAryabhAraM api pashyati
avAdIt yat vigata-chturvarSe kRSNAGgArasya utpAdane dasha-koTi-panJchAshAt-
lakSya-Tana-ityasya vRddhiH abhavat|

 
jammU-

aphagAna

aphagAnistAnasya uttarIya-kuMdUja-prAnte adya aphagAna-sainyabalAnAM
ArakSibalAnAM ch saMyukta-surakSA-saMrodha-sthaleSu tAlibAnasya AkramaNeSu
nyUnAtinyUnAH dasha-sainyabalAni mRtAni api ch SaT nAgarikAH mRtAH
santi| prAntasya adhikArijanaH avAdIt yat adya prAtaH e-jAla-janapade
surakSA-saMrodha-sthaleSu AkramaNaM abhavat| aphagAnistAnasya gajanI-prAnte
sthale gupta-visphoTena saha basayAnasya saMghaTanam‌ abhavat| asyAM
durghaTanAyAm SaT janAH mRtAH trayashch AhatAH jAtAH|
